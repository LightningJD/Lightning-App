2\. Design System

2.1 Typography

-   **DM Sans 600:** Profile name, page/section titles (Settings,
    Notifications headers)

-   **General Sans:** Body text, UI labels, buttons, handles, input text

-   **Playfair Display:** Testimony titles, pull quotes, avatar initial
    letters

2.2 Color System - Night Mode

-   **Background:** #0d0b18 (deep navy)

-   **Card/Glass:** rgba(255,255,255,0.04) bg + rgba(255,255,255,0.06)
    border

-   **Text Primary:** #e8e5f2

-   **Text Secondary:** #8e89a8

-   **Text Muted:** #5d5877

-   **Accent:** #7b76e0 (purple) / #9b96f5 (light purple)

-   **Notification Badge:** #ef4444 (red)

-   **Success:** #5cc88a

-   **Danger:** #e05c6c

2.3 Color System - Day Mode (Peri V3 Glass Edge)

-   **Background:** linear-gradient(135deg, #cdd8f8, #d6daf5, #dee0f6,
    #e4e0f5)

-   **Card/Glass:** rgba(255,255,255,0.5) bg + rgba(150,165,225,0.15)
    border + backdrop-filter:blur(12px)

-   **Text Primary:** #1e2b4a

-   **Text Secondary:** #4a5e88

-   **Text Muted:** #8e9ec0

-   **Accent:** #4facfe (blue) / #2b6cb0 (dark blue)

-   **Notification Badge:** #ef4444 (red, same both modes)

2.4 Glass Card Pattern

All cards, inputs, and interactive surfaces use a consistent glass
morphism pattern:

-   Night: rgba(255,255,255,0.04) background, rgba(255,255,255,0.06)
    border, no blur

-   Day: rgba(255,255,255,0.5) background, rgba(150,165,225,0.15)
    border, backdrop-filter: blur(12px)

-   Border radius: 10-14px for cards, 8-10px for inputs, 18px for pill
    inputs

-   Box shadow (day only): 0 1px 6px rgba(150,165,225,0.05)

3\. Global Behaviors

3.1 Navigation: Back Arrow + Swipe Right

Every page that is not a root tab gets a left arrow in the top-left
corner AND supports swipe-right gesture to go back. These are always
paired.

**Applies to:** Settings, Channel Chat, DM Conversation, Any User
Profile, Notifications Panel, Create Testimony (each step), Help Center,
Privacy Policy, Terms of Service, Blocked Users, Contact Support, any
sub-page pushed onto the navigation stack.

**Does NOT apply to:** Root tabs (Home, Charge, You) --- these are the
base level. Modals/dialogs (confirm, share, create server) --- these use
tap-outside or swipe-down to dismiss.

**Implementation:** Create a SwipeablePageWrapper component using
framer-motion or react-swipeable. Detect right swipe starting from left
30% of screen, threshold 80px. Wrap all pushable pages once.

3.2 Red Notification Badges (#ef4444)

All unread/active notification indicators use red #ef4444. Applied
consistently across:

-   Bottom nav badge (count on Home icon)

-   DM list badges (unread count per conversation)

-   Server rail dots (red dot on servers with activity)

-   Notification feed dots (red dot next to unread notifications)

-   Bell icon badge (count in header)

-   Channel list badges (unread count per channel)

3.3 Tab Rename: Find ‚Üí Charge

The middle tab is renamed from \'Find\' to \'Charge\' throughout the
entire app. The concept: reading testimonies charges your faith. Bolt
icon + \'Charge\' label. This rename applies to all references in code,
navigation, headers, and any user-facing text.

3.4 Colorless Outline SVG Icons

All UI icons across the app are colorless outline SVGs that inherit
color from their parent element. No colored emojis in the interface.
Icons change color only via active/inactive state.

**Icon replacements:**

-   Bell (notifications): Classic bell outline SVG

-   Chat (DM button): Double bubble outline SVG

-   Speaker (voice channels): Speaker + waves outline SVG

-   Server icons: Letter initial + gradient circle (default), image
    upload (optional)

-   Bottom nav Home: House + Tower combined outline SVG

-   Bottom nav Charge: Bolt outline SVG

-   Bottom nav You: Person outline SVG

3.5 Server Icon System

Server icons use letter initial on a gradient circle as the default.
Users can optionally upload a custom image.

-   Default: First letter of server name in DM Sans 700 on a gradient
    circle

-   8 gradient options to choose from when creating a server

-   Optional: Upload custom image (replaces letter+gradient)

-   This replaces the current emoji picker in CreateServerDialog
